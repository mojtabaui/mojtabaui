# کیت برند ملینا

## چی رو کجا استفاده کنم

| فایل | کاربرد |
|---|---|
| `melina-mark-*.svg` | فقط سیبیل، بدون قاب. برای وقتی خودت پس‌زمینه می‌سازی |
| `melina-square-*.svg` | سیبیل داخل مربع گردگوشه. نشان اصلی برند |
| `*-2000.png` | چاپ کوچک، اسلاید، شبکه‌های اجتماعی |
| `*-4000.png` | موکاپ، بیلبورد، هر جا که بزرگ می‌شه |

## کدوم رنگ

- `black` روی پس‌زمینه‌ی روشن
- `cream` روی پس‌زمینه‌ی تیره
- `violet` فقط برای تاکید، نه استفاده‌ی اصلی

## نکته

**SVG بی‌نهایت بزرگ می‌شه بدون افت کیفیت.** اگر ابزارت SVG قبول می‌کنه، همیشه
همون رو بده. PNGها فقط برای جاهاییه که SVG قبول نمی‌شه.

همه‌ی PNGها پس‌زمینه‌ی **شفاف** دارن.
